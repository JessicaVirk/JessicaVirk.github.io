Character Quiz

In this project, our task was to create a short quiz that would give the quiz taker a specific answer based on the choices they chose. Our requirements were to have a series of questions with more than one answer choice and these choices were supposed to result in more than one answer. Specifcally, I chose the topic of "Which Harry Potter Character Are You?" I provided a series of questions giving the user multiple responses they can choose from and gave the quiz taker up to four characters they can be chosen as. In the code, we had to use various features ranging from the click function all the way to numberous else if statements. I was able to learn how to properly format a click function and learned the advantage of using else if statements. It was interesting to know how these simple codes can immensely increase a web pages' accessibility.

Photo Gallery Project

In this project, our task was to create a series of images with an input box, allowing the user to input an image link for it to be shown on the page. The topic I chose was a list of books worth reading. I was able to use different CSS styles and explore various HTML functions and improve on such things as creating a functioning input box. I was able to really understand the importance of codes such as the (val) method that are all used to improve the user's experience.

Quote Generator

In this project, our task was to create a button that would allow the user to view different quotes that would be displayed at random. I was able to experiment with various CSS formats and work through my difficulties when creating a proper button. My main focus in this project was the visual appeal and I feel like with this project, I was really able to explore different alignments, colors and fonts.
